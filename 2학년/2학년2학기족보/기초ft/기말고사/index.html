중간고사 제출 문제

다음의 이미지를 기반으로해서 웹페이지를 완성해서 제출하면됩니다.

제출방법

  1. 이미지파일은 images/폴더의 것을 사용한다.
  2. html파일과 CSS파일 그리고 이미지 파일은 압축을 하여 파일명을 "학번_이름_제출일자.zip"으로 작성하여,
    deute.daums@daum.net 으로 메일을 보낸다.
  3. 메일 제목은 "기말고사 과제 제출 '학번' '이름'"으로 보낸다.
  4. 과제는 12월 18일까지 제출 완료한다. (도착 메일 기준 12월 18일 이후는 받지 않겠음)

중점 포인트

  1. 웹 접근성을 준수할 것
  2. 의미에 맞는 태그를 사용할 것
  3. 크로스브라우징이 될 것
